# docker-ztj1993-ssh-to-socks

ssh 转换为 socks5 服务

## 相关链接

## 使用教程

> 简单使用

```
docker run -ti --rm ztj1993/ssh-to-socks -e HOST=127.0.0.1 -e USER=root -p 123456
```
